Application

Ultimate Prompt Builder est un logiciel desktop gratuit, distribué sous licence open source.

Données personnelles

L'application fonctionne 100 % en local sur ta machine. Aucune donnée n'est transmise à Triskell Studio. Les clés API que tu configures sont stockées localement dans settings.json et ne sont transmises qu'au provider d'IA que tu choisis explicitement (Anthropic, OpenAI, Google, Mistral, xAI), selon leurs propres conditions d'utilisation.

Mises à jour

Les mises à jour sont vérifiées automatiquement via l'API publique GitHub. Aucune information sur ton usage n'est collectée à cette occasion.
